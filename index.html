Como crear tu propia casa inteligente

Si quieres hacer toda la programacion de tu casa domotica por tu cuenta es necesario que crees una aplicacion web o resapi para que te ayude a controlar todas las peticiones de los electrodomesticos inteligentes y demas, en mi caso utilice NodeJS para la programacion de esta. Puedes ver tutoriales en youtube.

Luego de tener tu propio servidor web puedes empezar a trabajar con los electrodomesticos con ayuda de arduinos o esp8266 con modulos wifi, si es que los electrodomesticos no vienen preparados para este tipo de proyectos.

El software Arduino de código abierto (IDE) hace que sea fácil escribir código y subirlo a la placa. Se ejecuta en Windows, Mac OS X y Linux. El entorno está escrito en Java y está basado en Processing y otro software de código abierto. Este software se puede usar con cualquier placa Arduino. Consulte la Introducción página de instrucciones de instalación.

El ESP8266 es un microchip Wi-Fi de bajo costo, con una pila TCP / IP completa y capacidad de microcontrolador, producido por Espressif Systems en Shanghai, China. El primer chip llamó la atención de los fabricantes occidentales en agosto de 2014 con el módulo ESP-01, fabricado por un fabricante de terceros, Ai-Thinker. Esta pequeña El módulo permite que los microcontroladores se conecten a una red Wi-Fi y establezcan conexiones TCP / IP simples utilizando comandos de estilo Hayes. Sin embargo, al principio hubo casi no hay documentación en inglés en el chip y los comandos que aceptó. El precio muy bajo y el hecho de que había muy pocas fuentes externas. Los componentes del módulo, que sugirieron que eventualmente podría ser muy barato en volumen, atrajeron a muchos hackers para explorar el módulo, el chip y el software que contiene, así como para traducir la documentación china.
